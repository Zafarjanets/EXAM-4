
# # 🛒 Техническое задание: Мини-интернет-магазин

# ## 🎯 Цель проекта

# Создать простое консольное приложение (или прототип) интернет-магазина с возможностью регистрации пользователей, авторизации, добавления товаров в корзину и совершения покупок. Все данные должны сохраняться в текстовых файлах.
# ## 📂 Структура проекта
# 1. **Классы**
#    * **User**
#      * Атрибуты:

#        * `username: str` – имя пользователя
#        * `password: str` – пароль
#        * `cart: list[Product]` – корзина (список выбранных товаров)
#      * Методы
#        * `register()` – регистрация пользователя (сохранение в `users.txt`)
#        * `login()` – вход пользователя (проверка данных в `users.txt`)
#        * `add_to_cart(product: Product)` – добавить товар в корзину
#        * `checkout()` – оформить заказ (уменьшает `stock` у товаров, очищает корзину, сохраняет историю покупок)
#    * **Product**
#      * Атрибуты:
#        * `name: str` – название товара
#        * `price: float` – цена товара
#        * `stock: int` – количество на складе
#      * Методы:

#        * `reduce_stock(amount: int)` – уменьшает количество товара на складе

#    * **Store**

#      * Атрибуты:

#        * `products: list[Product]` – список всех товаров в магазине
#      * Методы:

#        * `add_product(product: Product)` – добавить новый товар
#        * `remove_product(product: Product)` – удалить товар
#        * `find_product_by_name(name: str)` – найти товар по названию
#        * `list_products()` -   вывести список всех товаров

# ## 📑 Файлы данных

# * `users.txt`

#   * хранит список пользователей в формате:

#     ```
#     username:password
#     ivan:1234
#     anna:qwerty
#     ```
# * `products.txt`

#   * хранит список товаров в формате:

#     ```
#     name,price,stock
#     iPhone,1000,5
#     Laptop,1500,2
#     Headphones,100,10
#     ```
# * `history.txt`

#   * хранит историю покупок в формате:

#     ```
#     username: [товар1, товар2, ...]
#     ivan: [iPhone, Headphones]
#     anna: [Laptop]
#     ```

# ---

# ## 🔑 Основные функции

# 1. **Регистрация пользователя**

#    * Пользователь вводит `username` и `password`
#    * Проверка, что имя ещё не занято
#    * Сохранение в `users.txt`

# 2. **Авторизация**

#    * Пользователь вводит `username` и `password`
#    * Проверка соответствия данных в `users.txt`

# 3. **Просмотр товаров**

#    * Отображение всех товаров из `products.txt` (название, цена, количество в наличии)

# 4. **Добавление товара в корзину**

#    * Пользователь выбирает товар по названию
#    * Товар добавляется в `cart` пользователя

# 5. **Оформление покупки (checkout)**

#    * Проверка, что в корзине есть товары
#    * Уменьшение количества `stock` у товаров в `products.txt`
#    * Запись покупки в `history.txt`
#    * Очистка корзины

# 6. **Просмотр истории покупок**

#    * Пользователь может открыть свою историю из `history.txt`

# ---

# ## 🛠 Требования к реализации

# * Язык: **Python 3**
# * Хранение данных: **текстовые файлы** (`users.txt`, `products.txt`, `history.txt`)
# * Взаимодействие: консольное меню
# * Минимальный набор функций (MVP):

#   1. Регистрация и вход
#   2. Просмотр товаров
#   3. Добавление в корзину
#   4. Оформление заказа
